# CliniSight — Hospital Intelligence Platform

A full-stack backend platform for hospital operations management built with MySQL, FastAPI, and Streamlit.

## Tech Stack
- **Database**: MySQL
- **Backend**: Python · FastAPI · SQLAlchemy
- **Auth**: JWT Authentication
- **Caching**: Redis
- **Dashboard**: Streamlit · Plotly
- **Deployment**: Render · Supabase

## Database Schema
7 normalized tables: patients, doctors, departments, appointments, treatments, bills, rooms

## Features
- Normalized relational database with 7 interconnected tables
- Advanced SQL — JOINs, CTEs, Window Functions, Stored Procedures, Transactions, Views
- REST API backend with FastAPI
- JWT Authentication system
- Redis caching on analytics endpoints
- Interactive dashboard with real-time KPIs and charts
- Fully deployed with live URL
